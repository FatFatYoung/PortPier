# PortPier 更新日志

本项目的所有重要更改都将记录在此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2026-05-10

### 改进
- ✨ **语言切换体验优化** - 界面重建时自动继承运行日志
  - 解决了之前切换语言后日志丢失的问题
  - 服务端：主日志及防火墙日志（拦截/放行）均会刷新并保留历史
  - 客户端：连接日志平滑过渡，不再清空

### 技术改进
- 🔧 **UI 状态管理规范** - 引入 "Save-Destroy-Rebuild-Restore" 模式
- 🔧 **ScrolledText 状态处理** - 修复只读模式下无法插入历史文本的静默失败问题

## [1.1.0] - 2026-05-08

### 新增
- 🌐 **中英双语支持** - 完整的界面国际化
  - 服务端和客户端均支持中文/英文切换
  - 语言切换按钮显示目标语言
  - 切换语言后界面立即重建
  - 语言设置自动保存
- 📝 **日志消息国际化** - 所有连接日志支持中英文

### 修复
- 🐛 修复语言切换按钮显示逻辑
- 🐛 修复服务端设置页面滚动问题
- 🐛 修复首次运行配置文件路径问题
- 🐛 修复启动时双窗口问题

## [1.0.0] - 2026-05-04

### 新增
- ✨ 服务端可视化管理界面（tkinter GUI）
- ✨ 客户端可视化配置界面
- ✨ 多客户端同时连接支持
- ✨ 客户端认证机制
- ✨ 实时连接日志
- ✨ 防火墙日志查看
- ✨ IP 白名单/黑名单访问控制
- ✨ 自动重连机制
- ✨ 单文件 EXE 打包

### 技术特性
- 基于 Python asyncio 的异步架构
- TCP Keepalive 防止连接超时
- 64KB 缓冲区优化传输速度
- 支持 CIDR、IP 范围、通配符等 IP 格式
